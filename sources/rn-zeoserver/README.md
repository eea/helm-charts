# Reportnet Local Converters

The Reportek zeoserver

[See more](https://github.com/eea/eea.docker.reportek.zeoserver)

## Configuration

- `timezone` - Time zone.
- `storage` - Size of the persistent volume claim.
- `storageName` - Name of the persistent volume claim.
